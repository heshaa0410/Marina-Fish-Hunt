# Marina-Fish-Hunt

A small browser game where you start as a tiny fish, eat smaller fish to grow, earn points, and avoid the larger predator fish that can eat you.

## How to play
- Move with WASD or the arrow keys.
- Eat smaller fish to gain points and grow in size.
- Avoid larger fish.
- You have 10 lives total. Losing all lives ends the game.

## Run it locally
From the project folder, start a static server:

```bash
python3 -m http.server 8000
```

Then open:

http://localhost:8000
